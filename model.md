# 同好人格测试 — 五大模型 × 15维度

## 计分规则

每个维度包含 2 道题，每题 1~3 分，维度总分 2~6：
- 总分 2~3 → **L（低）**
- 总分 4   → **M（中）**
- 总分 5~6 → **H（高）**

---

## 模型一：热爱驱力模型（Passion Drive）

> 描述"为什么爱这个角色，爱的是什么形式"

### P1 沉迷深度

| 等级 | 描述 |
|------|------|
| L | 路人缘，刷到喜欢，不会主动追 |
| M | 有点上头，但还没影响日常 |
| H | 重度沉迷，脑子里常住，角色相关内容必追 |

### P2 深究度

| 等级 | 描述 |
|------|------|
| L | 喜欢就喜欢，不深挖人设和剧情 |
| M | 了解主线人设，但不专门去考据 |
| H | 喜欢考据、分析、研究角色的一切细节 |

### P3 拟人化程度

| 等级 | 描述 |
|------|------|
| L | 清楚是虚拟角色，欣赏但不过度投入情感 |
| M | 有时会把角色当作真实存在来感受 |
| H | 完全视角色为真实存在，为其开心/难过 |

---

## 模型二：产出创作模型（Creative Output）

> 描述"是否创作，如何创作"

### C1 创作冲动

| 等级 | 描述 |
|------|------|
| L | 纯消费者，只欣赏不产出 |
| M | 偶尔有想法，但很少付诸实践 |
| H | 热爱产出，创作是表达喜爱的主要方式 |

### C2 创作形式

同好群体的创作行为分为三类：

| 等级 | 描述 |
|------|------|
| L | 偏非实体创作向（绘图、同人文、视频剪辑、翻唱、同人音乐等，产出为数字/文字形式的作品） |
| M | 偏手工向（制作实体作品，素材可来自官方或他人的非实体创作，加工为配饰、挂件、纸制品、模型等） |
| H | 偏COS向（通过cosplay照片、视频及参加线下活动来表达对角色的喜爱） |

### C3 分享欲

| 等级 | 描述 |
|------|------|
| L | 内容只为自己留存，热衷建立私人档案库 |
| M | 自己存一份，也会转发分享 |
| H | 热衷传播，转发别人的内容比自己做的还积极 |

## 模型三：消费行为模型（Spending）

> 描述"如何用钱表达爱"

### Sp1 氪金意愿

| 等级 | 描述 |
|------|------|
| L | 零氪或微氪，爱是精神上的 |
| M | 选择性氪金，只为真正值得的内容付费 |
| H | 大氪，角色出了必支持，月均不看 |

### Sp2 周边购欲

| 等级 | 描述 |
|------|------|
| L | 不买周边，或只选一两件有纪念意义的 |
| M | 喜欢就买，但还在克制范围内 |
| H | 官方出啥买啥，同人谷子也不放过 |

### Sp3 线下参与

| 等级 | 描述 |
|------|------|
| L | 线上粉，从不参与线下活动 |
| M | 偶尔参与，条件合适才去 |
| H | 生日会、应援活动场场到，异地也冲 |

---

## 模型四：圈内社交模型（Community Social）

> 描述"如何与其他同好互动"

### So1 战斗性

| 等级 | 描述 |
|------|------|
| L | 佛系，遇到黑帖直接划走 |
| M | 该出手时出手，但不主动找事 |
| H | 遇黑必反，举报、回骂、带节奏，随时备战 |

### So2 社群活跃度

| 等级 | 描述 |
|------|------|
| L | 潜水党，只看不说 |
| M | 偶尔互动，有感触才出来说几句 |
| H | 活跃核心，发帖、组织、招募，圈子里的熟脸 |

### So3 从众性

| 等级 | 描述 |
|------|------|
| L | 独立思考，有自己判断，不随大流 |
| M | 参考大家意见，但保留自己看法 |
| H | 跟着大佬走，别人说怎么氪就怎么氪 |

---

## 模型五：同好身份模型（Fan Identity）

> 描述"如何定义自己的同好身份"

### Id1 专一度

| 等级 | 描述 |
|------|------|
| L | 多坑多推，喜欢的角色一大堆 |
| M | 有重点但不排斥其他 |
| H | 单推单坑，只有这一个角色在心里 |

### Id2 入圈标准

| 等级 | 描述 |
|------|------|
| L | 刷到喜欢就是同好，来者不拒 |
| M | 略有标准，但总体包容 |
| H | 有严格"真粉"标准，看不惯不够格的人 |

### Id3 圈外公开度

| 等级 | 描述 |
|------|------|
| L | 爱好严格保密，现实中没人知道 |
| M | 不主动说，问了也不否认 |
| H | 完全不藏，甚至主动安利别人入坑 |
