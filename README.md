# SLTI - Sylus's Lover Type Indicator

测测你是什么品种（划掉）类型的小狸花

小狸花 是《恋与深空》的角色 —— 秦彻 的心上人的昵称
不同的小狸花有不同的性格、爱好、消费习惯
通过 30 道场景题，从 5 大模型、15 个维度分析你的行为模式，匹配 26 种小狸花人格类型 （前端统称人格为品种）
本测试仅供庆祝秦彻0418生日快乐~

---

## 五大模型

| 模型 | 维度 |
|------|------|
| 热爱驱力 | 沉迷深度 · 深究度 · 拟人化程度 |
| 产出创作 | 创作冲动 · 创作形式 · 分享欲 |
| 消费行为 | 氪金意愿 · 周边购欲 · 线下参与 |
| 圈内社交 | 战斗性 · 社群活跃度 · 从众性 |
| 同好身份 | 专一度 · 入圈标准 · 圈外公开度 |

## 启动方式

### 方式一：直接打开
双击 `index.html` 用浏览器打开即可

### 方式二：本地服务器 (推荐)
```bash
# 进入项目目录
cd /Users/liying64/Downloads/ai/app_project/slti

# 使用 Python 启动
python3 -m http.server 8109

# 或使用 Node.js
npx serve .
```

然后在浏览器访问: http://localhost:8109
